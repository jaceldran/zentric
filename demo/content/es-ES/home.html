Demo Zentric

Este es el sitio demo de Zentric. Cada una de las opciones intenta mostrar alguna característica del funcionamiento.

Por ello, todas las páginas cargan un bloque de información contextual referida al proceso que se ha ejecutado para componer la respuesta. Esta funcionalidad puede servir también a para ayudar a depurar código durante el desarrollo.

Para mostrar/ocultar el panel con la información de contexto, pulsar el pequeño icono de información situado en el border derecho inferior de la página.

La información se muestra en un panel con 2 pestañas:

Route
Muestra la configuración de la ruta actual una vez se ha identificado y procesado.
  • app-template => Plantilla usada para la respuesta.
  • class => Clase o controlador.
  • method => El método de la clase anterior que es llamado para componer la respuesta.
  • uri => La uri recibida.
  • pattern => El patrón identificado en el router.
  • params => Lista de parámetros recibidos en la uri, si la hay.
Method
Muestra el código del método vinculado a la ruta que se ha ejecutado para componer y enviar la respuesta.
